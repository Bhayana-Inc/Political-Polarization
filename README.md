# Political Polarization on X (Twitter)

This repository contains analysis code for our **Political Polarization** project, which examines how influencers, politicians, and user accounts shaped and amplified polarization on X (Twitter) during the **2024 U.S. Presidential Election**. The focus is on network structure, engagement patterns, and content signals (hashtags, sentiment, etc.) across partisan communities.
